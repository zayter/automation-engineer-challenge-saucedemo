### Setup Instructions 

#####  Create virtual environment with the following steps:

  - `pip3 install virtualenv`
  - `virtualenv mypython`
  - `source mypython/bin/activate`
  
  
##### Install requirements  

- `pip3 install -r requirements.txt` 

##### Execute any features with behave:

`behave features/ `
